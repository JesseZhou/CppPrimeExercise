ch8复习题
	4、现有结构
	  struct box
	  {
	    char maker[40];
	    float height;
	    float width;
	    float length;
	    float volume;	
	  };

	  a、编写函数，以box的引用为形参，显示每个成员的值

	  错解：
	    void method1( struct box & b1 )
	   {
		cout << b1.maker << b1.height << b1.width << b1.length << b1.volume << endl;
	   }		

	   正解：
	     因为只需要展示数据，不需要处理数据，所以入口参数定义为const类型
	    void method1( const struct box & b1 )
	   {
		cout << b1.maker << b1.height << b1.width << b1.length << b1.volume << endl;
	   }

	ch10:
		2 封装，抽象，隐藏是什么
			错解：	
				抽象：how?我不知道
				封装：提供成员函数，给用户使用
				隐藏：设成员变量的访问域为private,禁止用户直接访问
				
			正解：
				抽象：在类里，用户可通过公用接口来执行对类对象的操作。这是抽象在类里的体现（解释：抽象是概括，抽象可用于获取对象的属性和功能）
				
				隐藏：类的私有数据成员只能通过成员函数来访问
				
				封装：用户不知道类操作的实现细节，是封装

		8 什么是默认函数，默认函数的好处是
			错解：	
				定义：用户未定义类构造函数时，编译器生成的默认构造函数
				好处：给成员变量赋初值
			
			正解：	
				定义：是没有参数或所有参数都有默认值的构造函数
				好处：创建对象时可不用显式地初始化它。在定义对象数组时不用初始化。
